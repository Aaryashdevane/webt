Basic HTML Text Formatting

This is a simple demonstration of text formatting using HTML.

Commonly used text formatting tags:

  • <b>: Bold Text
  • <strong>: Strong Emphasis
  • <i>: Italicized Text
  • <em>: Emphasized Text
  • <u>: Underlined Text
  • <s>: Strikethrough Text
  • <mark>: Highlighted Text
  • <small>: Smaller Text
  • <sub>: H2O (Subscript)
  • <sup>: x2 (Superscript)
  • <blockquote>:
    This is a blockquote, typically used to display quoted text.
  • <pre>: 
    This is preformatted text
It preserves    both spaces and line breaks.
  • <code>: Inline code example

Each of these tags serves a specific purpose in formatting your text content.
